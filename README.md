# Arduino Trivia Processor

This is a fun trivia game made with the Arduino and Processor made from Processor

## Materials

- Breadboard
- 2 LED
- Arduino Uno
- Around 20 jumper wires
- 5 220ohm resistors
- 3 Buttons
